# Java多线程多进程生产者-缓存队列-消费者模式

## 简介
在自己的PC机上使用多进程，多线程实现生产者-缓存队列-消费者模式，进程间通过socket通信。

## 使用的技术

- 多进程通信：socket
- 多线程：Thread
- 缓存队列：redis (java [jedis](https://github.com/xetorthio/jedis))
